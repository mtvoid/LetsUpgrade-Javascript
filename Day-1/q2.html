var, let, and const

JavaScript variables can be declared via the var, let, and const keywords. The latter two were introduced into the language with ES6

var: Variables declared with var are global in scope. They can be declared anywhere in the code, and they can be re-declared, as well as their values can be modified.

let: Variables declared with let have a block scope, (i.e., code enclosed in braces {}). Like with var, their value can be modified, but the variables cannot be redeclared within the same scope, and trying to do so will generate an error.

const: Variables declared with const also have a block scope. In addition, they can neither be redeclared, nor can their values be modified.

Declaration type Scope Can be re-declared Value can be changed
var Global Yes Yes
let Block No Yes
const Block No No
