JavaScript console methods

The JavaScript console can be used to display output from one's JavaScript code. This can be used for logging information, and especially useful for debugging.

The console object comes with various methods to log different types of information.

console.log(): This is useful for general logging of information to the console. It will accept any type of variable passed to it as a parameter.

console.info(): For logging informative text. The output is printed to indicate this.

console.warn(): For logging warnings (such as non-critical errors that need to be brought to attention). The output is printed with a warning label and colour.

console.error(): For logging errors (such as more severe problems that might prevent the code from running correctly). The output is printed with an error label and colour.

console.table(): This is useful for logging tabular data, that can be passed to this method in the form of an array or an object.

console.time() and console.timeEnd(): These methods can be used to run a timer. Used together, they will log the time it takes to execute one or more JavaScript statements between the start and end of the timer methods.
